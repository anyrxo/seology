    • Slide-up text reveals on hover

  MODULE 3: Link Button Hover Animations
    • Same mechanism as nav links
    • Applied to footer links
    • Applied to CTA buttons
    • Smooth vertical slide effect

  MODULE 4: Service Card Accordions
    • Height-based expand/collapse
    • Initial state: height: 0px
    • Hover: expands to content height
    • 0.4s ease transition

  MODULE 5: Scroll Reveal Animations
    • IntersectionObserver API (better performance than scroll events)
    • Fade in: opacity 0 → 1
    • Slide up: translateY(40px) → 0
    • 0.8s cubic-bezier(0.22, 1, 0.36, 1)
    • Applied to: sections, cards, features

  MODULE 6: Webflow IX2 Integration
    • Full compatibility with Webflow's IX2 engine
    • Preserves all data-w-id attributes
    • Maintains IX2 animation styles in <style> block

  MODULE 7: Smooth Scroll
    • Native browser smooth scroll
    • Applied to all anchor links (href="#...")
    • Automatic for back-to-top button

  MODULE 8: Back to Top Button
    • Appears after scrolling 500px
    • Fade in/out with opacity transition
    • Smooth scroll to top when clicked

  TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════

  Issue: Animations not working at all
    → Check browser console for errors
    → Hard refresh (Ctrl+Shift+R)
    → Verify webflow.js is loaded
    → Try different browser

  Issue: Frame images not animating
    → Check if .frame-image elements exist on page
    → Verify inline styles are present
    → Check for CSS conflicts

  Issue: Scroll reveal not triggering
    → Check element selectors in HTML
    → Test IntersectionObserver support
    → Adjust rootMargin in script

  Issue: Performance problems
    → Close other browser tabs
    → Disable browser extensions
    → Check Performance tab in DevTools
