## INTERACTION STATES

### Buttons
**Idle:**
- Primary: Blue gradient (blue-600 â†’ purple-600)
- Secondary: Gray border, transparent fill

**Hover:**
- Primary: Lighter gradient, scale(1.05), shadow glow
- Secondary: Gray fill, white text

**Active/Click:**
- Subtle scale down (0.98)
- Shadow reduces

### Cards
**Idle:**
- Border: Gray-800
- Background: Gray-900

**Hover:**
- Border: Blue-500/50 (semi-transparent)
- Shadow: Blue glow
- Scale: 1.02
- Transition: 300ms ease

### FAQ Accordions
**Collapsed:**
- Arrow pointing right (â–¶)
- Border bottom only

**Expanded:**
- Arrow pointing down (â–¼)
- Full border
- Content slides down (300ms)
- Background slightly lighter

## SCROLL ANIMATIONS

All sections use **Intersection Observer** or **Framer Motion**:

### Pattern
1. Element enters viewport
2. Fade opacity: 0 â†’ 1 (600ms)
3. Slide up: translateY(50px) â†’ 0 (600ms)
4. Stagger children by 100ms

### Specific Elements
- **Hero:** Immediate (no animation)
- **Social Proof Stats:** Count up animation
- **How It Works Steps:** Stagger left to right
- **Feature Cards:** Stagger in grid order
- **Testimonials:** Fade in one at a time
- **Pricing Cards:** Slight bounce on entrance
